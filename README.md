NFS World Boss v1

Files:
- index.html
- styles.css
- app.js
- README.txt

Local test:
1. Open this folder in VS Code.
2. Right-click index.html and choose "Open with Live Server".

Alternative:
  python -m http.server 8000 (FRONTEND)
  npm run dev (BACKEND)
Then open:
  http://localhost:8000

Notes:
- No default layers are included.
- Layer ID is separate from Layer 1 / Layer 2 / Layer 3.
- Layers are sorted by ID.
- Data is stored locally in browser localStorage in this backup version

Functions:
- Able to send discord messages through user input
- Discord Login
- User input data creates schedule
- Schedule is saved and can be further manipulated
